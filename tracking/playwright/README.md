## Layout

```text
/Users/xin/issue-tracking/playwright/
  <project>/
    <branch>/

        html-report/

```

## Naming

- `project`: repo directory name, for example `copy_names`
- `branch`: current git branch, replace `/` with `_`
- `issue`: prefer Feishu `record_id`, else ticket id, else short manual slug
