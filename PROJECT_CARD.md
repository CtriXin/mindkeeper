# PROJECT_CARD — MindKeeper

## 一句话
Personal Cognitive Substrate：跨 AI、跨设备的持续认知底座，真正懂你的 AI

## 核心亮点

### 1. SCE 对外 / PCS 对内
- **SCE** (Symbiotic Cognitive Entity) = 产品叙事，你的 AI 伙伴
- **PCS** (Personal Cognitive Substrate) = 系统真相，持续演化的认知底座
- 不再说"一个真正活着的 AI"，而是说"一套 continuity contract"

### 2. 6 类认知对象
不是只有"notes"，而是分层：
- Evidence → Observation → Belief → Procedure → Policy → Thread Capsule

### 3. 7 段学习回路
从"记住"升级到"持续演化"：
- Capture → Distill → Reflect → Promote → Compile → Act → Maintain

### 4. substrate/avatar 分离
- Claude/Codex/GPT 不是"同一个脑的不同半球"
- 而是"共享 substrate 的不同 avatar surface"
- 共享：thread capsule, beliefs, policies, receipts
- 不共享：推理轨迹、局部风格、failure mode

### 5. Continuity Contract
明确承诺什么、不承诺什么：
- ✅ 承诺：memory / thread / project / policy / cross-surface continuity
- ❌ 不承诺：完整人格恒定、情绪连续、所有模型输出一致

### 6. 后台预热
从 Context-Gateway 学到：
- 不等用户问了才检索
- 75% context 时后台开始预热/压缩
- 预测性加载 > 反应性加载

## 可分享的金句
