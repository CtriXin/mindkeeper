## MASTERCLASS CONSTRAINTS
1. **ANTI-CRYPTO**: No glowing orbs/radial gradients; No glassmorphism/blur. Background = #000000 (Pure Black).
2. **NO CONTAINER**: Sharp corners (radius:0); 1px hard borders; No soft shadows.
3. **BROKEN GRID**: Typography over images; Negative margins; Break section boundaries.
4. **90/10 COLOR**: 90% Pure White/Black, 10% High-saturation accent (Neon). No 50/50 balance.
5. **HYPER-CONTRAST**: H1 > 120px; Body text < 45ch width. No medium font sizes.
6. **PHYSICS**: Custom magnetic cursor; Spring physics for hover (No ease-in-out); Environmental awareness (time-based decay).
7. **METADATA**: 10px Mono tags (INDEX, GEO, TIMESTAMP) as decor.
8. **NAV DEFIANCE**: No "Home/About"; Use opinionated labels; Dispersed corner layout.
9. **KINETIC**: Infinite marquee; Scroll-velocity typography distortion.
