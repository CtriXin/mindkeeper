# MindKeeper 8 个工具怎么用？这篇讲清楚

<!-- 封面图：docs/cards/03-tools.png -->

---

MindKeeper 是给 AI 编程助手加持久记忆的 MCP Server。

装上后 AI 多出 8 个工具，分三大类。

---

## 🟢 知识管理

<!-- 插图1：docs/cards/03-tools.png 截取知识管理部分，或直接用完整图 -->

**brain_learn** — 提取经验
完成一个有价值的任务后，AI 自动结构化为 Recipe（步骤/文件/坑点）。

**brain_recall** — 语义搜索
「懒加载」「延迟加载」「lazy load」都能匹配到同一条。55+ 同义词组 + Trigram 模糊。

**brain_list** — 浏览知识库
按项目 / 标签 / 框架过滤。显示健康统计（活跃/过时/已降权）。

---

## 🟡 上下文蒸馏

**brain_bootstrap** — 冷启动
恢复上次 Thread + Board 信号 + 相关 Recipe 推送。一个工具搞定所有冷启动。

**brain_checkpoint** — 保存进度
蒸馏当前状态：决策/变更/发现/待续，写入 Thread 文件。

**brain_threads** — 查看线程
按 repo 分组显示所有活跃 Thread。

---

## 🔵 项目看板

**brain_board** — 读写看板
四象限紧急/重要矩阵 + 备忘。每个项目独立一个看板。

**brain_check** — 信号扫描
自动检测 Deadline 临近 / 逾期 / 僵尸 item / Recipe 健康。

---

## 日常使用流程

```
开始工作 → brain_bootstrap（自动恢复 + 推送）
  ↓
写代码 → brain_recall（遇到类似问题时搜索）
  ↓
完成任务 → brain_learn（提取经验为 Recipe）
  ↓
阶段结束 → brain_checkpoint（蒸馏状态）
  ↓
下次继续 → brain_bootstrap（30秒恢复）
```
