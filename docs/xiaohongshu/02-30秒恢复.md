# AI 编程最烦的事：「昨天做到哪了？」

<!-- 封面图：docs/cards/04-30s.png -->

---

每天早上打开 Claude Code 的第一件事：

「继续昨天的工作。」

然后 AI：「你好！有什么可以帮你的？」

你花 10 分钟手动复述昨天的上下文。
一周就是一个多小时。一个月就是一整天。

**MindKeeper 的解法：蒸馏 + 恢复，全自动。**

---

## 下班前：一键蒸馏

AI 调用 `brain_checkpoint`，自动提取：

📋 **决策**：用 JWT 替代 session cookie
📁 **变更**：auth.ts, middleware.ts, config.ts
💡 **发现**：iOS Safari 需要 polyfill
⏭️ **待续**：部署验证 + 写测试

保存为一个 Thread 蒸馏文件，持久化到本地。
不是保存整个对话，是**蒸馏出关键信息**。

几百字就能恢复一个完整的工作上下文。

---

## 第二天：30 秒恢复

<!-- 插图1：docs/cards/04-30s.png（如封面相同则换 30s-recovery.excalidraw 导出图） -->

打开新 Session，AI 调用 `brain_bootstrap`：

1️⃣ 恢复昨天的 Thread → 所有上下文回来了
2️⃣ 推送相关 Recipe → 「JWT + Safari 有已知方案」
3️⃣ Board 信号提醒 → 「auth 重构周四前完成」

**30 秒，直接进入工作状态。** 不是让你重新解释，是 AI 自己就知道该做什么。

---

## 不会越积越多

Thread 文件有 TTL（默认 7 天），过期自动 GC。
保持整洁，不会变成另一个垃圾场。
